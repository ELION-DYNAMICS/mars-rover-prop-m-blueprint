# Mars Rover Prop-M Blueprint

## Mission
Reconstruct and modernize the PrOP-M class Mars rover architecture with:
- Physically accurate dynamics
- Testable autonomy stack
- Reproducible simulation
- Verification-driven development

## Modes
- PROP-M Mode (tether-limited, cyclic science execution)
- Modern Autonomous Mode (Nav2-based)

## Engineering Philosophy
- Simulation-first
- Logging mandatory
- Regression tested
- Deterministic scenarios

v0.1 Simulation MVP
v0.2 Navigation baseline
v0.3 PROP-M behavior loop
v1.0 Terramechanics validation
